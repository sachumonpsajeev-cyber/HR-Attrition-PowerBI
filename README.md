# HR Attrition Analysis Portfolio

## 📈 Project Objective
This HR Analytics dashboard was designed to transform raw workforce data into actionable intelligence. The core goal was to identify **predictive indicators of employee turnover** and provide HR leadership with a visual tool to prioritize retention strategies for high-risk roles and departments.

## 🛠️ Technical Stack & Skills
* **Tooling:** Power BI Desktop
* **Data Transformation:** Power Query (M Language) for data cleansing, handling null values, and normalizing inconsistent dataset fields.
* **Analytical Modeling:** Custom **DAX (Data Analysis Expressions)** development for KPI generation and trend analysis.
* **Visualization:** Dashboard design focused on "Storytelling with Data" (Executive Summary → Drill-down Details → Driver Correlation).

## 🚀 Key Analytical Findings
The analysis uncovered four critical levers for retention:
1.  **Overtime Impact:** Employees working overtime are **3x more likely** to leave, signaling a potential work-life balance crisis.
2.  **Age Vulnerability:** The 18–25 age bracket shows an attrition rate of **34.8%**, indicating a possible "onboarding or growth" gap.
3.  **Compensation Gap:** Terminated employees earned an average of **$2,000 less/month** than active staff, highlighting salary as a primary friction point.
4.  **Role-Specific Risk:** Sales Representatives are the most volatile segment with an attrition rate of **39.8%**.
